# Decision Tree from Scratch

This project implements a Decision Tree classifier from scratch using Python and NumPy.

## Concepts implemented

- Entropy calculation
- Information gain
- Recursive tree building
- Node splitting
- Prediction using tree traversal
- Decision boundary visualization
- Accuracy vs tree depth analysis

## Dataset

Synthetic dataset generated using sklearn.make_classification

## How to run

python decision_tree.py

## Output

- Model accuracy
- Decision boundary visualization
- Accuracy vs depth graph
- Prediction visualization
